# محفظتي الشخصية - Ibrahim's Portfolio

موقع شخصي بسيط وجميل يعرض معلوماتي ومشاريعي في علوم الحاسوب.

A simple and beautiful personal website showcasing my information and computer science projects.

## المميزات | Features

✨ تصميم حديث وسريع الاستجابة | Modern and responsive design
🎨 ألوان أزرق جميلة | Beautiful blue color scheme
📱 صديق للهواتف | Mobile-friendly
🔧 سهل التخصيص | Easy to customize

## البدء | Getting Started

1. استنسخ المستودع | Clone the repository
2. افتح `index.html` في المتصفح | Open `index.html` in your browser
3. عدّل المحتوى حسب احتياجاتك | Customize the content as needed

## هيكل الملفات | File Structure

```
portfolio/
├── index.html    # الملف الرئيسي | Main file
├── styles.css    # أنماط التصميم | Styling
└── README.md     # التوثيق | Documentation
```

## التخصيص | Customization

### تغيير المعلومات الشخصية | Change Personal Information

عدّل النصوص في `index.html` وأضف:
- بريدك الإلكتروني | Your email
- روابط وسائل التواصل | Social media links
- مشاريعك الفعلية | Your actual projects

### تغيير الألوان | Change Colors

عدّل متغيرات CSS في `styles.css`:

```css
:root {
    --primary-blue: #0066cc;     /* اللون الأساسي | Primary color */
    --dark-blue: #003d99;        /* اللون الداكن | Dark color */
    --light-blue: #e6f0ff;       /* اللون الفاتح | Light color */
}
```

## نشر الموقع | Deploy the Website

### GitHub Pages

1. اذهب إلى إعدادات المستودع | Go to repository settings
2. انتقل إلى GitHub Pages
3. اختر `main` كفرع المصدر | Select `main` as source branch
4. سيكون موقعك متاح على: `https://Ibrahdl515.github.io/portfolio`
